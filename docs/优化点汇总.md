## 二、待实现的 AI 优化点

### 🔧 角色③-1：搜索词飞轮（Phase B + C）
- **做什么**：从采集的标题中，AI 自动提取候选搜索词 → 去噪 → 去重 → 五维评分 → 入队
- **价值**：词库自我膨胀，从 10 个种子词 → 85 个精准词
- **成本**：每轮 ~¥0.3 API 费用
- **依赖**：智谱 API（已有 key）

### 🔧 角色③-2：标题素材库
- **做什么**：噪声词（闲置/自用/99新/包邮）不丢弃，按场景归类存储
- **价值**：上架时 AI 自动推荐标题组合，提高点击率
- **成本**：零额外 API 调用（复用 Phase B 的输出）

### 🔧 角色③-3：PDD 搜索词优化
- **做什么**：闲鱼标题 → AI 提取核心产品词 → 生成 4 种策略的 PDD 搜索变体 → 评估最优策略
- **价值**：提高货源匹配精准度，减少无效搜索
- **成本**：每次货源查找 +1-2 次 AI 调用

### 🔧 角色②：品类评分进化
- **做什么**：积累 5+ 条预测 vs 实际利润数据 → AI 分析维度相关性 → 自动调整模型权重
- **价值**：每个品类长出自己最准的评分公式
- **成本**：每品类每 5 条数据触发一次 AI 分析

### 🔧 控制机制
- **品类重叠处理**：AI 语义去重，合并搜索结果重叠 >70% 的词
- **偏见循环检测**：主导品类 >50% 自动提权其他品类
- **队列上限**：待采集词 >80 自动降级
- **自动降级**：30 天无产出 → 过期淘汰
